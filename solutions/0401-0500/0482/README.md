**(№ 482) License Key Formatting [easy]**

Даны строка s и число k. Строка s это лицензионный ключ. В нем есть только буквы, цифры и символ `-`. Нужно отформатировать строку по следующим правилам:
1. разделить все символы на группы, в которых k символов, кроме первой группы, в ней может быть меньше символов
2. между группами нужно поставить символ `-`
3. все буквы сделать заглавными

**Решение на JavaScript**

```javascript
var licenseKeyFormatting = function(s, k) {
 let str = '';
  
  for (let i = s.length - 1; i >= 0; i--) {
    if (s[i] !== '-') {
      str += str.length % (k + 1) === k ? '-' : '';
      str += s[i];
    }
  }
  
  return str.split('').reverse().join('').toUpperCase();
};
```

**Объяснение решения**

Заведем строку str в которой будем формировать новую отформатированную строку. Пойдем циклом по всей исходной строке с конца и будем проверять каждый символ является `-` или нет. Если нет, тогда мы проверяем какая на данный момент длина строки str. Если она делится на k + 1 без остатка, значит мы уже сформировали нужное количество символов в группе и закрыли ее символом `-`, поэтому дальше формируем следующую группу. Если делится с остатком то следим, когда остаток будет равен k значит следующим символов ставим `-` потому что мы отсчитали нужное количество символов. Когда таким образом сформируем все группы у нас будет перевернутая строка в переменной str. Разворачиваем ее и делаем все буквы заглавными.
