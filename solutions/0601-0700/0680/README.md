**(№ 680) Valid Palindrome II [easy]**

Дана строка **s**. Нужно узнать может это строка стать палиндромом, если удалить из нее только один символ или ни одного. Если может вернуть true, если нет вернуть false.

**Решение на JavaScript**

```javascript
var validPalindrome = function(s) {
  let left = 0;
  let right = s.length - 1;

  while (left < right) {
    if (s[left] !== s[right]) {
      let one = s.slice(left, right);
      let two = s.slice(left + 1, right + 1);

      return one === [...one].reverse().join('') || two === [...two].reverse().join('');
    }

    left++;
    right--;
  }

  return true;
};
```

**Объяснение решения**

```javascript
var validPalindrome = function(s) {
  // (1) Заводим два курсора left и right. При первом запуске left указывает на первый символ,
  // right указывает на последний символ
  let left = 0;
  let right = s.length - 1;

  // (2) Проверяем не указывают ли курсоры left и right на один и тот же символ. Наша задача
  // отбросить одинаковые символы слева и справа и найти только ту часть строки, где символ
  // слева не совпадает с протиположным ему символом справа, потому что это то самое место,
  // где можно попробовать удалить один символ из строки и проверить, после удаления
  // строка получится в валидный палиндром. Поэтому если курсоры left и right не указывают
  // на одинаковый символ мы сначала проверяем не совпадают ли символы по этим курсорам (*)
  // и если совпадают, тогда мы передвигаем курсоры на символы поглубже, left на один символ
  // вперед, right на один символ назад (**)
  while (left < right) {
    // (*)
    if (s[left] !== s[right]) {

      // (3) Если мы все-таки нашли два не одинаковых символа по краям рассматриваемой части
      // строки, тогда мы в переменной one сохраняем часть строки без последнего символа,
      // а в переменной two сохраняем часть строки без первого символа. То есть отрезаем
      // по одному символу
      let one = s.slice(left, right);
      let two = s.slice(left + 1, right + 1);

      // (4) И смотрим если хоть в одной переменной one или two получились такие части
      // строки, что если их полностью перевернуть каждую слева направо и они будут
      // совпадать со своей неперевернутой версией, значит из строки можно удалить один
      // символ и из нее получится валидный палиндром, возвращаем true. Если ни одна
      // из перевернутых частей не совпадет со своей неперевернутой версией значит
      // возвращаем false 
      return one === [...one].reverse().join('') || two === [...two].reverse().join('');
    }

    // (**)
    left++;
    right--;
  }

  // (5) Если строка изначально была пустой или состояла из одно символа, значит
  // никаких проверок делать не надо, считаем, что она валидный палиндром
  return true;
};
```
