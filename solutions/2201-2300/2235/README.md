**(№ 2235) Add Two Integers [easy]**

Дано два целых числа. Нужно найти их сумму и вернуть ее.

**Решение на JavaScript**

```javascript
var sum = function(num1, num2) {
  return num1 + num2;   
};
```

**Объяснение решения**

Складываем два числа и сразу возвращаем.
