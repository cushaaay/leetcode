**(№ 977) Squares of a Sorted Array [easy]**

Дан массив чисел `nums`, отсортированный в неубывающем порядке. Нужно вернуть массив квадратов каждого числа, отсортированный в неубывающем порядке.

**Решение на JavaScript**

```javascript
var sortedSquares = function(nums) {
  let res = [];
  let left = 0;
  let right = nums.length - 1;

  while (left <= right) {
    if (nums[left]**2 > nums[right]**2) {
      res.push(nums[left]**2);
      left++;
    } else {
      res.push(nums[right]**2);
      right--;
    }
  }

  return res.reverse();
};
```

**Объяснение решения**

```javascript
var sortedSquares = function(nums) {
  // (1) Заводим переменную res, в которую будем складывать квадраты чисел
  let res = [];

  // (2) В массиве есть отрицательные числа, поэтому может получиться так, что
  // самое маленькое отрицательное число после возведения в квадрат станет больше,
  // чем самое большое положительное число возведенное в квадрат. Поэтому нам нужно
  // сравнивать крайние числа и для этого мы заводим переменные left и right.
  // В самом начале эти указатели будут указывать на первое и последнее числа.
  // Но когда мы определим какое из них больше в квадратной форме, это число больше
  // рассматривать не нужно, поэтому мы будем передвигать указатели на соседние числа
  let left = 0;
  let right = nums.length - 1;

  // (3) Запускаем цикл до тех пор, пока оба указателя не встретятся на одном числе
  while (left <= right) {

    // (4) Возводим в квадрат самое маленькое число и самое большое
    if (nums[left]**2 > nums[right]**2) {

      // (5) Если квадрат маленького числа оказался больше квадрата большого числа,
      // тогда помещаем квадрат маленького числа в переменную res и передвигаем
      // указатель left на одно число вперед. Теперь самое маленькое рассматриваемое
      // число другое
      res.push(nums[left]**2);
      left++;
    } else {

      // (6) Если квадрат большого числа оказался больше квадрата маленького числа,
      // тогда помещаем квадрат большого числа в переменную res, а указатель right
      // передвигаем на одно число назад
      res.push(nums[right]**2);
      right--;
    }
  }

  // (7) Когда все числа будут пройдены и все квадраты добавлены, они будут лежать
  // в переменной res в невозрастающем порядке, а нам надо наоборот,
  // в неубывающем порядке, поэтому делаем реверс и возвращаем ответ
  return res.reverse();
};
```
